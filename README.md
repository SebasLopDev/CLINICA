## 📌 Descripción del Proyecto

Sistema web hecho en **Flask** para la gestión de una clínica médica. Permite registrar pacientes, médicos, agendar citas y almacenar historiales clínicos.

## 🚀 Tecnologías Utilizadas

- Python 3.x
- Flask
- HTML / CSS
